ExplicitEulerStep

Defaults

algorithm="euler" performs a single-stage, order-1 forward-Euler update under fixed-step control. Forward Euler carries no embedded error estimate, so adaptive control is unavailable, and as an explicit method it needs no nonlinear or linear solver. See Default settings for the defaults shared across algorithms.
